# MOA Bodychart PWA v1.1.5

MOA BUSAN의 통증 바디차트 · 자세·정렬 관찰 웹앱입니다.

## v1.1.5 변경사항

- Chrome 데스크톱에서 차트 위 마우스 휠/트랙패드 스크롤이 막히던 문제 수정
- v1.1.4의 강제 `wheel` 이벤트와 `preventDefault()` 제거
- Chrome과 Safari 모두 브라우저 기본 스크롤 동작 사용
- 모바일 터치 입력과 차트 클릭 기능 유지
- v1.1.3 전면 정렬 방향 수정 유지
