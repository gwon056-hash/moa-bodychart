- 데스크톱 Chrome의 바디차트 입력 이벤트를 `click` 전용으로 분리

- 차트 위 마우스 휠·트랙패드 스크롤을 브라우저 기본 동작에 완전히 위임

- 기존 평가 저장·검색·백업·전면 정렬 수정 기능 유지


- v1.1.7: Chrome 데스크톱 휠 스크롤 호환성 개선(overscroll reset + non-blocking fallback).
