## 데이터 저장 방식

저장된 평가는 브라우저의 로컬 저장소에 보관됩니다. 서버로 자동 전송되지 않습니다.

- `평가 저장`: 현재 평가를 저장 목록에 추가합니다.
- `평가 업데이트`: 저장 목록에서 연 평가를 수정한 뒤 같은 평가에 반영합니다.
- `저장 목록`: 환자 이름/코드, 날짜, 메모로 검색합니다.
- `전체 데이터 백업`: 저장 평가 전체와 현재 작업을 JSON 하나로 저장합니다.
- `전체 백업 복원`: 기존 평가를 지우지 않고 안전하게 병합합니다.

공용 기기에서는 환자 실명보다 이름/코드 사용을 권장합니다. 앱 삭제, 브라우저 데이터 삭제, 기기 변경 전에는 전체 백업을 권장합니다.
